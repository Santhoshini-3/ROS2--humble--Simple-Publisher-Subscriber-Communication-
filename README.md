# ROS-Simple-Publisher-Subscriber-Communication-
The ROS2 Python scripts below define a publisher node that sends a message every 0.1 seconds (at 10Hz frequency) and a subscriber node that logs each received message. The publisher broadcasts a running count appended to a string, showcasing basic real-time communication between nodes in ROS.
